# Glitch Better Have My Money 1/3

Ce challenge est le premier d'une série de 3 challenges, réalisés par Philippe_Katerine, qui se suivent mais sont résolvables indépendamment les uns les autres. 
Ce premier challenge est classé en très facile : il s'agit de lire une datasheet PDF fournie d'un composant électronique et d'en extraires les informations.
