# Glitch Better Have My Money [2/3]

Ce challenge est le deuxième d'une série de 3 challenges, réalisés par Philippe_Katerine, qui se suivent mais sont résolvables indépendamment les uns les autres.
Ce deuxième challenge est classé en moyen : il s'agit d'effectuer une attaque par injection de faute en utilisant un chip whisperer virtuel accessible via une api (/run_code).
Les participants doivent prendre en main leur lab en connectant le voltmètre et l'adaptateur uart correctement et une intéragir avec le chipwhisperer via l'API.
Le but est de glitcher la séquence de boot du device pour skipper l'instruction qui désactive le port de débug UART, afin de démarrer la caméra tout en ayant un débug UART.
