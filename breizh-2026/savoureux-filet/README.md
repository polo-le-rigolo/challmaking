# Savoureux Filet

Ce challenge est un challenge radio / réseau réalisé par Philippe_Katerine, et classé en *Facile*. 
Les joueurs doivent déchiffer des communications meshtastics à partir d'un dump (message envoyé d'un node à un autre, capturé via l'API meshtastic.) Ces messages sont envoyés en longfast canal par défaut, avec la clé de chiffrement par défaut.
