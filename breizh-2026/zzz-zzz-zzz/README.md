# Zzz Zzz Zzz 

Ce challenge est un challenge radio, réalisé par Philippe_Katerine.

Le challenge reposait sur la faille classique de Zigbee : l'utilisation de la clé d'appairage par défaut de la Zigbee Alliance.

Une fois la capture déchiffrée, il suffisait d'observer les commandes `Move to Level with OnOff` afin de récupérer les niveaux de luminosité envoyés à l'ampoule et de les interpréter comme des codes ASCII.
